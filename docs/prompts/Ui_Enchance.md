**In Dark Mode, the Menu button is difficult to see. The current design uses white text on a gray background, which creates low contrast and poor visibility.
Please adjust the color contrast so the Menu button is more visible and accessible in Dark Mode (for example, by using a darker background or a more contrasting text color).
**The "Logout button" is displayed in the menu dropdown even when you are not logged in.
**The menu dropdown is transparent, so when it opens it overlaps with other UI elements in both desktop and mobile views. It should have a solid background and proper layering.
Also, when the dropdown menu is open, clicking outside of it does not close it. It would be better if the dropdown automatically closed when the user clicks outside.
After logging in, a green horizontal line blinks above the bottom navbar along with the Home page blink, which causes a small change in the scrollbar size each time it blinks. This layout shift should be fixed.
In mobile screen sizes, the Add button on the Home and Activity pages is hidden behind the bottom navbar, so the layout should be adjusted to keep the button fully visible.
Additionally, in mobile view, the "Press enter to continue" text at the bottom of the Groups page is partially cut off, so it should be moved up or spaced properly to be fully visible.
**In pop-up windows such as Create Group and Add Expense, please remove the top-center drag slider used to close the pop-up by dragging it down, as it is not necessary.
Also, on the Account Page, the background colors of the three pills (Friends, Paid, and Total) should be adjusted so the text remains clearly visible in Dark Mode with better contrast.
